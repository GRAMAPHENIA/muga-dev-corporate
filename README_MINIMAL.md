# Documentación mínima

## Proyecto: Cabañas Patagonia

Sitio web desarrollado con Astro, Tailwind CSS y TypeScript para alquiler de cabañas en la Patagonia argentina.

- **Framework:** Astro
- **Estilos:** Tailwind CSS
- **Lenguaje:** TypeScript
- **Version:** 1.0.0

### Estructura principal
- `src/components/`: Componentes reutilizables (home, about, pricing, contact, layout)
- `src/pages/`: Páginas principales del sitio
- `public/cabins/`: Imágenes de las cabañas
- `layouts/Layout.astro`: Layout base con transiciones

### Scripts
- `pnpm dev`: Inicia el servidor de desarrollo
- `pnpm build`: Compila el sitio para producción
- `pnpm preview`: Previsualiza el sitio compilado

### Personalización
- Cambia textos e imágenes en los componentes dentro de `src/components/`
- Configura estilos en `tailwind.config.cjs`
